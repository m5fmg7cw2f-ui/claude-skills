# Scientific Paper Structure (Math) Skill

## Purpose
Guide Claude to structure a mathematics paper correctly, following conventions of top journals and arXiv standards.

## Trigger
Use this skill when the user asks to:
- Outline or structure a math paper
- Organize sections of a paper
- Check if a paper has the right structure
- Plan the narrative flow of a mathematical article

## Standard Structure

### 1. Title and Abstract
- Title: precise, contains key mathematical objects
- Abstract: 150-250 words; state the main result explicitly; no citations in abstract

### 2. Introduction
- Motivate the problem (1-2 paragraphs)
- State the main results informally but precisely
- Explain the key ideas and novelty
- Describe the organization of the paper
- End with notation conventions if needed

### 3. Preliminaries
- Define all objects used in the paper
- State without proof background results that will be used
- Establish notation

### 4. Main Results
- Organize by logical dependency, not chronology
- Each section should have a clear goal stated at its start
- Lemmas before theorems that use them
- Include proof sketches for major theorems if proofs are long

### 5. Applications and Examples
- Concrete examples illustrating the main results
- Connections to other areas

### 6. Conclusion and Open Problems
- Summarize what was proved
- List natural open questions

### 7. References
- Complete, consistent bibliography in standard format

## Narrative Principles
- The paper should tell a story with a clear arc
- Every section should have a stated purpose
- A reader should be able to read Introduction plus statements of results and understand the contribution
